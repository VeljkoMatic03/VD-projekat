# VD-projekat
Simple JavaScript Tetris based game playable in browser, made as a university project

To play, just clone this repo and open tetris-uputstvo.html.
There you can select which types of shapes are included in the game, and the starting difficulty level of the game.
After that you can click the "Zapocnite igru" button to start the game.
Game can be played by arrow buttons on keyboard, or by using buttons on the webpage.
After the Game Over screen you will be redirected to the statistics screen.
All previous high scores are being saved in browser local storage.
